Proyecto 1: Coche a control Bluetooth

Control de coche con Arduino

En este proyecto repare un coche radio control descompuesto por un sistema BT ocupé el espacio 3

del motor para la dirección del coche y el espacio 1 para la fuerza del coche

Materiales

  • Arduino Uno
  • Motor shield L293D
  • Motores (en mi caso, 2)
  • HC-05 o HC-06
  • Alguna batería capaz de dar el voltaje y amperaje que necesites
