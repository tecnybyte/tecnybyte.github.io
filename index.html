Proyecto 1: Coche a control Bluetooth

Control de coche con Arduino

En este proyecto repare un coche radio control descompuesto por un sistema BT ocupe el espacio 3

del motor para la direccion del coche y el espacio 1 para la fuerza del coche

Materiales

  • Arduino Uno
  • motor shield L293D
  • motores (en mi caso 2)
  • hc-05 o hc-06
  • alguna bateria capas de dar el voltage y amperage que necesites
